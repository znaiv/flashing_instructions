# VoltageOS Sunny Flashing Guide - Next.js Voltage Edition

A Vercel-ready Next.js + TailwindCSS flashing guide for Xiaomi Redmi Note 10 `sunny/mojito`.

## Features

- Next.js App Router
- TailwindCSS
- Framer Motion animations
- Glass UI
- Copy buttons
- Mobile responsive navigation

## Local setup

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```
